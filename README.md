**Cobalt Investment Explorer**

A Java and SQL prototype for exploring U.S. cobalt mining investment opportunities using USGS data. Includes a normalized relational database, custom SQL stored procedures for commodity filtering and data cleaning, and a Java GUI for interactive data exploration.
